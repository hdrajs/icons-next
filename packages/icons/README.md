# icons-next

This package contains Free Community 7000+ icons converted to React SVG Icon components.

## Installation

The icons-next package install both with the following command:

```bash
npm install icons-next
```
Using yarn
```bash
yarn add icons-next
```
Using pnpm
```bash
pnpm add icons-next
```

## Contributing

The Icons package is updated via a script extracts the SVG elements from there. Because of this, we accept new icons that diverge from the source from svg file.

This process is performed by the maintainers on a quarterly basis.
